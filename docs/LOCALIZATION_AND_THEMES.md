# Localization and Themes

The Flutter UI ships with:

- Languages: Russian, English
- Theme modes: System, Light, Dark

## Onboarding

The first launch experience asks for:

- Language
- Theme mode

Selections are persisted locally and can be changed later in Settings without reinstalling the app.

## Accessibility

- All screens must remain readable in Light and Dark mode.
- Status banners use explicit iconography plus text.
- Log viewer preserves contrast and allows severity filtering.
